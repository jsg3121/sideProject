# 1달에 1개의 프로젝트 만들기

1달에 하나씩 간단한 페이지부터 CRUD, 페이징, 검색기능 등이 들어간 화면까지 작게작게 만들어가며 저장하는곳 입니다.

프로젝트는 주로 vue.js로 만들 예정이며 velog에서 프로젝트를 만들어가는 과정을 게시글로 확인하실 수 있습니다.
